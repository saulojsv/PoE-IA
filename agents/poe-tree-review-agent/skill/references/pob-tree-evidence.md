# PoB tree evidence

Use these as maintained discovery references; verify the installed PoB version and current PoE patch before relying on behavior.

- Path of Building Community repository: https://github.com/PathOfBuildingCommunity/PathOfBuilding
- Community site and tree-planning overview: https://pathofbuilding.community/
- PoB keyboard/mouse shortcuts, including path-trace mode: https://github-wiki-see.page/m/PathOfBuildingCommunity/PathOfBuilding/wiki/Keyboard-%26-Mouse-Shortcuts

Relevant operational facts: PoB is an offline build planner; it imports passive-tree links and its tree view supports path tracing. These facts support efficient exploration, but node legality, calculations, and patch behavior must be checked in the actual installed build.
