# Local PoE Build Knowledge

Base local para o agente entender como builds funcionam a partir de XMLs do PoB/poe.ninja e regras extraidas de forum.

- `build_notes/`: nota JSON e Markdown por build.
- `build_knowledge.jsonl`: todas as notas em JSONL para treino/RAG.
- `skill_rules.json`: padroes por skill.
- `class_index.json`: builds por classe.
- `ascendancy_index.json`: builds por ascendencia.
- `forum_rules.json`: regras gerais de interpretacao vindas do forum.
