Hi, thanks!

1) To get started you can use uncorrupted gear, but have you checked the prices?

- Corrupted 90% res multi helmet: ~20c
- +1 curse ammy: ~1d
- Vulnerability gloves: ~50c
- Spell Supp boots: ~1d
- %int belt: ~30c

2) We don't have a huge Life pool, so nasty degens like those dark circles that Cortex drops will kill you quick! But we can get Chaos Res capped, bleed/CB immune, have an Instant Recovery Life Flask, Enduring Cry Regen, and 10% Instant Leech, so as long as you can hit something, and you're not standing in it forever, degens are not an insurmountable problem to deal with, just don't stand in shit ;)

3) Blight is annoying cuz you gotta be pretty much melee range to take out some of the big/thick baddies, but if you can freeze/stun them with empower, DPS should be fine. Breach is np, but Splitting Doesn't have whole screen clear. Ultimatum, I don't do/like much, but it's mostly about staying alive, so if I were to tweak the build for it, I am sure it could run them, but you'd need someone that knows the mechanic better than I do. Legion, again kinda same issue as Breach: Splitting doesn't have whole screen clear. Heist is fine, very zoomy! Yes, Delving is very fun! Deepest I usually go in a league is ~400, but from there I'm sure you can tweak to keep going.

4) I don't see why you couldn't boss rush with this. It's definitely fast enough to get to the bosses, and probably go with a Single Target DPS setup focus, like what you can see in the 3.22 Sanctum PoB Export. Though, I haven't seen the cringe thumbnail ur talkin bout lol!

Seliot wrote:
could you elaborate on crafting clusters? They're kinda pricy and if I can get them cheaper by crafting it would be cool


You buy:

1 x Large Cluster with "adds 8 passive skills" + "12% inc dmg with 2H weapons"

2 x Medium Clusters with "adds 4 passive skills" + "10% inc proj dmg"

All preferably between ILVLs 50-67 (a bit pricier, but they eliminate a lot of possible rolls when higher ILVL). If you go higher ILVL you can still craft them, but you'll spend more currency doing so.

Small clusters you can buy for a few chaos, so not worth crafting.

To craft:

Trans > Alt till one or two of the good mods > Aug for a second good mod, if only got one > If failed to get second good mod, Alt + Aug over and over till you have 2 good mods > Regal and pray for third good mod > Scour and start over if failed.

I'm sure you can also use Grave Crafting for these too, but you'll need to look into youtube videos to figure that out.
