@Everyone: If you're having problems upgrading, it's because exalted-chaos rate is pretty low(1-62 at the moment) this league. This makes items worth more exas on Trade(a little more than double) because an exa used to worth 140-150c around these times in past leagues. It's best to do runs targeted to get many chaoses. Such as doing delve, since this league niko rotas are back in global 820 chat and delve is a good way to farm chaoses from many wealth chests. You will also get high level heist contracts in delve to do them, and heists are a good way to get chaoses fast as well. Can make many exas per day this way. Chaos recipe will work fine too, and also heist contracts dropping there. These 3 can be done altogether in a day.

Hinsy wrote:
Hi, I really can't paste the build in POB. Any clue whY?

Hi. For some reason this league some players are having problems importing from pastebin. But this is a rare occasion. I just recreated both gears' Pobs from scratch and updated Pob links with those using latest version of Community Fork Pob, hoping that it might help if problem was in Pob links. If they still don't work, make sure you're using Community Pob;

https://github.com/PathOfBuildingCommunity/PathOfBuilding/releases

Make sure you're using the latest version and use zip version(using zip version solved the problem for some players). If still not working, there is an alternative solution. First enter the pastebin link on your web browser. Click clone on top then copy the whole text. Then on Pob, paste the copied text to the box on the left side of "Import from Pastebin", where it says "To import a build, enter the code here:". This should work without problems.
