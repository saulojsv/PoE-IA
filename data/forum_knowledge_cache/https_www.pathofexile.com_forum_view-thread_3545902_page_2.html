I have not yet tried Simulacrum with this build, but you should not lack damage as the skill can regularly inflict 30m+ total impales with crits, which you can then pull or manually consume.

For the pantheons, for regular mapping I use Lunaris for physical damage mitigation, since the build uses Abyssus which increases physical damage taken by quite a lot, and Ralakesh for the same reason.

Alternatives would be Solaris for doing single bosses and Arakaali for fights where you're worried about degens.
