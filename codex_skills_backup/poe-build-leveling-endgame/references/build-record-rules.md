# Build Record Rules

Build classifications:
- league_starter
- budget
- mid_budget
- high_budget
- mirror_tier
- mapping
- bossing
- all_rounder
- hardcore
- ssf
- ruthless
- beginner
- advanced
- off_meta
- meme
- experimental

Required fields:
```json
{
  "build_id": "",
  "name": "",
  "main_skill_id": "",
  "patch": "",
  "league": "",
  "class": "",
  "ascendancy": "",
  "budget_tier": "",
  "content_focus": [],
  "strengths": [],
  "weaknesses": [],
  "defensive_layers": [],
  "damage_scaling": [],
  "gem_links": [],
  "gear": [],
  "passives": [],
  "leveling": [],
  "pob_links": [],
  "video_links": [],
  "guide_links": [],
  "author": "",
  "source_date": "",
  "verification_status": ""
}
```

Do not say "best build" without criteria.
