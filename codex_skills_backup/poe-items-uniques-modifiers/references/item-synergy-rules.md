# Item Synergy Rules

Evaluate every item/mod as:
- `core`
- `strong`
- `optional`
- `niche`
- `historical`
- `unverified`

Capture:
```json
{
  "item_or_mod": "",
  "slot": "",
  "synergy_type": "direct|scaling|defensive|utility|enabler|market",
  "reason": "",
  "importance": "",
  "patch": "",
  "source_url": "",
  "confidence": 0.0
}
```

Check uniques, rares, implicits, eldritch implicits, influenced/fractured/synthesized/crafted/veiled mods, corruptions, jewels, clusters, flasks, weapons, shields, armour, helmets, gloves, boots, rings, amulets, belts, quivers.
