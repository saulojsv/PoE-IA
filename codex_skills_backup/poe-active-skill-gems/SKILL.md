# PoE Active Skill Gems

Use for non-support active skill gems in the A-Z queue.

Collect: identity, aliases, variants, tags, requirements, weapon/equipment restrictions, costs, cooldown, levels, quality, damage/effectiveness, duration, radius, projectiles, crit, mechanics, formulas, breakpoints, links, supports, items, passives, ascendancies, builds, guides, PoB, market/meta, bugs, patch changes, sources.

Save only the current skill:
- `data/normalized/skills/{normalized_name}.json`
- `data/exports/skills/{normalized_name}.md`

Never infer compatibility from tags alone. Mark missing facts `unverified`.

When category tags apply, combine with the specialized skill: minion, trigger, reservation, guard, movement, warcry, brand, trap, mine, totem, curse, mark, herald, stance, retaliation, or channeling.

Preferred local sections in the skill JSON: `identity`, `official_stats`, `mechanics`, `scaling`, `links`, `supports`, `items`, `tree`, `builds`, `sources`.
