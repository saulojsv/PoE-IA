# PoE Support Gems

Use for support gems and support compatibility sections.

Classify each relationship: `compatible`, `incompatible`, `conditional`, `technically_compatible_but_ineffective`, `deprecated_interaction`, or `unknown`.

Each relationship needs support_id, skill_id, effect_summary, conditions, patch, sources, confidence, and validation status. Never assume support behavior from shared tags alone.

Also record why a support is bad or ineffective when popular guides avoid it. Mark this as `creator_opinion` or `community_consensus`, not official fact.
